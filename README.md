**Setup Steps**

1. open terminal type: "npm i" to install dependencies
2. to start server type: "npm run start:dev"
3. open localhost:3000 in browser search bar

** common psql cmds for reference**

1. list dbs : \l
2. use db : \c <name_of_db>
3. list tables : \dt
